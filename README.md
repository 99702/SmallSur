### Quick install 
Run `./install.sh` with `sudo` to install the theme
Set wallpaper from Wallpaper folder

### Install plank
#### For Debian
```
sudo apt update
sudo apt install plank -y
```
#### For Arch
```
sudo pacman -Syu
sudo pacman -S plank -y
```
